# Java Practice Playground 🎮

Zbiór rozwiązań zadań przygotowawczych do kursu MAS (Modelowanie i Analiza Systemów 
Informacyjnych) na temat podstaw programowania w języku Java.

## 📋 Zakres tematyczny:
- Tablice i kolekcje (List, Set, Map)
- Generatory tekstu i liczb
- Programowanie obiektowe (klasy, dziedziczenie, interfejsy)
- Obsługa plików (ZIP, try-with-resources)
- Java 8+ features (streams, lambdy, Optional, var)
- Refaktoryzacja i dobre praktyki
- Wydajność kolekcji

## 🔧 Wymagania:
- Java 11 lub nowszy (ze względu na var i inne features)
- IntelliJ IDEA (polecana) lub inne IDE

## 🚀 Uruchomienie:
Każde zadanie to osobna klasa z metodą main - można uruchamiać niezależnie.
